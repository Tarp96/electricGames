Welcome to the Electric Game API

Hello Electric Games, thank you for giving us the task to set up your API!
This API contains information about various different vidoe games. About every game in the database you will find
information about the games:

  • Numerical ID
  • Title
  • Platform
  • Year of Release
  • Image

Our Endpoints

Currently we have 3 endpoints that allow you to get information from the database. To make changes to our database, use our Electric Game app!

  • Get All the Information

    This link allows you to see all the information in our database
